About The Project

Camuflaged is an interactive and user-friendly interface designed to enhance the student experience while navigating attendance systems. Built with simplicity and efficiency in mind, it provides an alternative UI that makes the process of logging in and marking attendance seamless, engaging, and distraction-free.

Key Features:

✅ Familiar Login Process – A clean, intuitive UI that simplifies authentication.
✅ QR Code Scanning – Quick and hassle-free attendance marking.
✅ Minimalist & Engaging Design – A student-focused interface for a smoother experience.
✅ Performance-Oriented – Ensures a faster, more fluid attendance workflow.

Note: Camuflaged does not modify or interfere with any official system—it simply provides a modernized interface that enhances ease of use for students. It operates within existing workflows, ensuring compliance with institutional standards while giving users a more engaging and optimized way to manage their attendance.

Disclaimer: Camuflaged is an independent, student-driven UI enhancement tool. Users remain responsible for compliance with institutional policies.
